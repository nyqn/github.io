  3. 基本概念

    1. 版本库：.git隐藏文件夹，里面存储了许多配置信息，日志信息和文件版本信息等

    2. 工作目录（工作区）：与.git同级的全部文件夹，也是.git的父文件夹，用于存放代码

    3. 暂存区（stage）：.git文件夹中的index文件（binary）

      工作区——git add——>暂存区——git commit——>版本库

  4. 工作目录下文件的两种状态

    • untracked：没有被纳入版本控制

    • tracked:纳入版本控制后的细分状态
